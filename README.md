## Discord bot
### Invitation
The Bot only needs the following permissions:
- Send Messages
- Add Reactions
- Use External Emoji
- Manage Messages

To quickly add the bot to your server, click [here](https://discord.com/api/oauth2/authorize?client_id=827207000005541909&permissions=272448&scope=bot)
